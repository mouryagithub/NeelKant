# Meeting Notes

Store team meeting notes in this directory.

## Naming Convention

Use the format: `YYYY-MM-DD_meeting_topic.md`

Example: `2026-05-21_project_kickoff.md`

## Template

```markdown
# Meeting: [Topic]
**Date:** YYYY-MM-DD
**Attendees:** Name1, Name2, Name3
**Duration:** X hours

## Agenda
1. Item 1
2. Item 2

## Discussion
- Point 1
- Point 2

## Decisions
- Decision 1
- Decision 2

## Action Items
- [ ] @person — Task description (due date)
- [ ] @person — Task description (due date)

## Next Meeting
- Date: YYYY-MM-DD
- Topic: [Topic]
```
